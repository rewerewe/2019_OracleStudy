git

(1) 원격과 로컬의 버전을 맞춰야 한다.
그러니까 로컬을 원격에 맞추어야 한다.
로컬의 (커밋: .....[d777714]) 이 부분과 
원격(git hub)의 Latest commit d777714 이 같은지 확인하자.

(2) 만약 둘의 버전이 맞지 않아 pull 과 push 가 되지 않는다면
로컬의 폴더를 날려. 탭도 꺼준다.
그리고 원격(git hub)의 clone or download 에서 복사한 주소로
New tab - Clone 해주자.  소스경로 / url 에 붙여넣었으면 
목적지 경로:는 내가 원하는 곳으로 설정해 로컬(.git) 폴더 만들어 주면 됨.
(폴더명은 원격 폴더명과 똑같이 적어주자)

(3) 그럼 로컬 저장소에 폴더, 파일들이 쭈욱 복사 될거야.

(4) 원격 git hub 새로고침 : 저장소 - 원격상태새로고침
    로컬 저장소 새로고침 : F5
